# void

## What is the Void

A place where the focus is on what's happening right now and removing everything else.

## How To Get Started

### Requirements

1. Postgresql
2. Java 21
3. Maven
4. NPM

### Interesting Tools

1. pgAdmin 4 - for Postgresql


### Building API

```mvn clean package```

Set SPRING_ACTIVE_PROFILE to "local" in your environment variables
